# 物理ノート (Physics Notes)

物理の勉強した内容をわかりやすく説明していくGitHub Pagesサイトです。

## サイトについて

このリポジトリは、物理学の学習内容を体系的に整理し、わかりやすく説明するためのノートサイトです。

### 掲載内容

- **力学**: 運動法則、エネルギー保存則など
- **電磁気学**: クーロンの法則、電場、磁場など
- **熱力学**: 理想気体の状態方程式、熱力学法則など
- **量子力学**: 光電効果、波動と粒子の二重性など

## ローカルでの実行

### 必要な環境

- Ruby 3.1以上
- Bundler

### セットアップ

```bash
# 依存関係のインストール
bundle install

# ローカルサーバーの起動
bundle exec jekyll serve

# ブラウザで http://localhost:4000 にアクセス
```

## デプロイ

このサイトはGitHub Actionsを使って自動的にGitHub Pagesにデプロイされます。

`main`ブランチにプッシュすると、自動的にビルドとデプロイが実行されます。

## コンテンツの追加

### 新しいトピックの追加

各分野のディレクトリに新しいMarkdownファイルを追加します：

- 力学: `_mechanics/`
- 電磁気学: `_electromagnetism/`
- 熱力学: `_thermodynamics/`
- 量子力学: `_quantum/`

### ファイルのフォーマット

```markdown
---
title: トピックのタイトル
layout: page
---

# トピックのタイトル

内容をここに記述...
```
