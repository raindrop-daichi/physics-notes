# 理想気体の状態方程式

理想気体の状態方程式は、気体の圧力、体積、温度の関係を表す重要な式です。

## 状態方程式

```
PV = nRT
```

### 記号の意味

- P: 圧力（パスカル、Pa）
- V: 体積（立方メートル、m³）
- n: 物質量（モル、mol）
- R: 気体定数（8.31 J/(mol·K)）
- T: 絶対温度（ケルビン、K）

## わかりやすく説明すると

この式は、気体の「圧力」「体積」「温度」の3つが、どのように関係しているかを表しています。

### 圧力と体積の関係（温度一定）

**ボイルの法則**: P × V = 一定

- 体積を半分にすると、圧力は2倍になる
- 体積を2倍にすると、圧力は半分になる

**例**: 注射器のピストンを押すと、中の空気の体積が減り圧力が上がる

### 体積と温度の関係（圧力一定）

**シャルルの法則**: V / T = 一定

- 温度を上げると、体積が増える
- 温度を下げると、体積が減る

**例**: 熱気球は、空気を温めることで体積を増やし、浮力を得る

### 圧力と温度の関係（体積一定）

**ゲイ=リュサックの法則**: P / T = 一定

- 温度を上げると、圧力が上がる
- 温度を下げると、圧力が下がる

**例**: 密閉容器を温めると、中の気体の圧力が上がる

## 理想気体とは？

### 理想気体の仮定

1. **分子の大きさは無視できる**: 気体分子自体の体積はゼロ
2. **分子間力は無い**: 分子同士は力を及ぼし合わない
3. **完全弾性衝突**: 衝突でエネルギーは失われない

### 実在気体との違い

実際の気体は理想気体ではありませんが：

- **低圧・高温**では理想気体に近い振る舞いをする
- 日常的な条件下では、理想気体として扱っても問題ない

## 日常の例

### タイヤの空気圧

- **夏**: 気温が高い → タイヤ内の圧力が上がる
- **冬**: 気温が低い → タイヤ内の圧力が下がる
- 定期的な空気圧チェックが必要な理由

### ポテトチップスの袋

- **高地**: 気圧が低い → 袋が膨らむ
- **平地**: 気圧が高い → 袋が萎む
- 山に持っていくと袋がパンパンになる

### エアコン

- 気体の圧縮と膨張を利用
- 圧縮すると温度上昇、膨張すると温度低下
- 状態方程式の実用例

## 絶対温度（ケルビン）

### セルシウス温度からの変換

```
T(K) = t(℃) + 273.15
```

### 重要な温度

- **絶対零度**: 0 K = -273.15℃（理論上の最低温度）
- **水の凝固点**: 273.15 K = 0℃
- **水の沸点**: 373.15 K = 100℃

### なぜ絶対温度を使うのか？

- 温度と体積・圧力の関係が比例関係になる
- 物理法則を簡潔に表現できる

## 応用問題の考え方

### 状態変化を追う

1. **初期状態**: P₁、V₁、T₁
2. **最終状態**: P₂、V₂、T₂
3. **関係式**: P₁V₁/T₁ = P₂V₂/T₂

### よくある問題パターン

- 一定温度で体積を変化 → 圧力を求める
- 一定圧力で温度を変化 → 体積を求める
- 一定体積で温度を変化 → 圧力を求める

## まとめ

理想気体の状態方程式は、気体の振る舞いを理解する基本です。この式を使えば、圧力・体積・温度の3つの量のうち、2つがわかれば残りの1つを計算できます。
