# エネルギー保存則

エネルギー保存則は物理学における最も重要な法則の一つです。

## 基本原理

> 孤立系における総エネルギーは常に一定である。エネルギーは形を変えることはあっても、新しく作られたり消えたりすることはない。

## エネルギーの種類

### 運動エネルギー（Kinetic Energy）

物体が運動することによって持つエネルギー

```
E_k = (1/2)mv²
```

- m: 質量（kg）
- v: 速度（m/s）

### 位置エネルギー（Potential Energy）

物体が位置によって持つエネルギー

重力による位置エネルギー：
```
E_p = mgh
```

- m: 質量（kg）
- g: 重力加速度（9.8 m/s²）
- h: 高さ（m）

## わかりやすい例

### 振り子の運動

振り子は運動エネルギーと位置エネルギーの相互変換の良い例です：

1. **最高点**: 速度ゼロ → 運動エネルギー最小、位置エネルギー最大
2. **最下点**: 速度最大 → 運動エネルギー最大、位置エネルギー最小
3. **途中**: 両方のエネルギーを持つ

どの位置でも、運動エネルギー + 位置エネルギー = 一定

### ジェットコースター

- 高い位置: 位置エネルギーが大きい
- 下降中: 位置エネルギーが運動エネルギーに変換
- 低い位置: 速度が速い（運動エネルギーが大きい）

## 実生活での応用

### 水力発電

1. ダムの高い位置に水を溜める（位置エネルギー）
2. 水を落とす（位置エネルギー → 運動エネルギー）
3. タービンを回す（運動エネルギー → 電気エネルギー）

### 再生可能エネルギー

- 太陽光発電: 光エネルギー → 電気エネルギー
- 風力発電: 風の運動エネルギー → 電気エネルギー

## 重要なポイント

1. **エネルギーは保存される**: 形が変わるだけで消えない
2. **実際には摩擦がある**: 熱エネルギーとして散逸する
3. **全体では保存**: 熱も含めれば総エネルギーは保存される

## 数式での表現

力学的エネルギー保存則：
```
E_k + E_p = 一定
```

変化の式：
```
(1/2)mv₁² + mgh₁ = (1/2)mv₂² + mgh₂
```

## まとめ

エネルギー保存則は、物理現象を理解する上で非常に強力なツールです。この法則を使えば、複雑な運動も簡単に解析できることがあります。
